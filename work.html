MeetUp App

Description

Meetup is an app to view the events in your desired city and it can be downloaded to your desktop or mobile!

Technologies & Methodologies Used

It is a Create-React-App, developed thru Test-Driven Development, Behavior-Driven Development approach and tested using Jest, Jest, Cucumber & Puppeteer frameworks. It has been designed and developed with Advanced Development concepts such as Serverless functions using AWS Cloud provider, Progressive Web App and Data Visualization.
